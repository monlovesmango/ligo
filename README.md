# create virtual environment
```
python3 -m venv .venv
```
```
source .venv/bin/activate
```

# install dependencies
```
pip install -r ./requirements.txt 
```

# load gwplot.py
``` 
python -i gwplot.py```

```

# start it up
```
start()
```
